# FON Gallery

图片画廊平台，支持图片上传、审核、分享等功能。

## 功能

### 公开页面
- **画廊首页** - 浏览所有已审核通过的图片，支持搜索和筛选
- **图片详情** - 查看图片详细信息、相关推荐和作者作品

### 用户功能
- **登录/注册** - 支持邮箱登录和 GitHub OAuth 登录
- **个人空间** - 管理自己上传的图片
- **团队空间** - 团队协作管理图片
- **设置** - 修改个人资料、头像和密码

### 管理功能
- **管理员后台** - 图片审核、批量抓取、数据统计

## 技术栈

Vue 3 + Vite + Vue Router + Tailwind CSS

## 开发

```bash
npm run dev
```
